# ズーム

**⌃1**（Control+1）を押すとズームモードに入ります。画面がキャプチャされ、ズームイン・アウトやパンが可能です。

## 操作

| 入力 | アクション |
|---|---|
| マウス移動 | パン |
| スクロールホイール / ↑↓ | ズームイン / ズームアウト |
| クリック | ドローモードに入る（ズーム中の画面が描画キャンバスになります） |
| Escape | ズームモードを終了（ドローから入った場合はズームに戻る） |
| 右クリック | ズームモードを終了 |

## ズーム → ドロー → ズームの流れ

ズームモード中にクリックすると、ズーム中の画面の上でドローモードに入ります：

1. **ズームモード** — クリックでドローに入る
2. **ドローモード**（ズームされたキャンバス上） — **Escape** でズームに戻る
3. **ズームモード** — もう一度 **Escape** で完全に終了

この2段階解除は、ドロー内のテキストモードと同様の動作です。

## ライブズーム

**⌃4**（Control+4）を押すとライブズームモードに入ります。通常のズームと異なり、画面の内容がリアルタイムに更新されます — 動画、ターミナル、アニメーションがズーム中も再生され続けます。

### 操作

| 入力 | アクション |
|---|---|
| マウス移動 | ズームされた表示領域をパン |
| スクロールホイール / ↑↓ | ズームイン / ズームアウト（1.0x〜8.0x） |
| クリック | フレームを静止してドローモードに入る |
| Escape | ライブズームを終了 |
| 右クリック | ライブズームを終了 |

### スティルズームとの違い

| | スティルズーム（⌃1） | ライブズーム（⌃4） |
|---|---|---|
| 画面の内容 | 静止スナップショット | リアルタイム更新 |
| マウスカーソル | 写らない場合がある（スナップショットに含まれる） | 常に表示 |
| ドローに入る | クリック（ズーム維持） | クリック（現在のフレームを静止してから描画） |
