# 使い方

各機能はグローバルホットキーまたはメニューバーのアイコンから起動できます。

| 機能 | デフォルトホットキー |
|---|---|
| ズーム | **⌃1**（Control+1） |
| ライブズーム | **⌃4**（Control+4） |
| ドロー | **⌃2**（Control+2） |
| 休憩タイマー | **⌃3**（Control+3） |

::: tip
ホットキーは**設定**でカスタマイズできます（メニューバーアイコン → 設定）。
:::

## 機能

### [ズーム](./usage/zoom)

画面をキャプチャし、スクロールでスムーズにズームイン・アウト。マウス移動でパン。ワンクリックでドローモードへ。

### [ライブズーム](./usage/zoom#ライブズーム)

画面をリアルタイムに更新しながらズーム — 動画、ターミナル、アニメーションがズーム中も再生され続けます。クリックでフレームを静止してドローモードへ。

### [ドロー](./usage/draw)

フリーハンド描画、図形（直線、矩形、楕円、矢印）、テキスト入力、複数の色、蛍光ペンモード — すべてキーボードショートカットで操作。

### [休憩タイマー](./usage/break-timer)

プレゼンテーションや休憩用の全画面カウントダウンタイマー。時間調整可能、色のカスタマイズ対応、バックグラウンド動作。
